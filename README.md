<!--  -->
1. Firstly create the virtual environment by hitting command 'python3 -m venv venv' in terminal. Then activate that with the help of 'source venv/bin/activate' command this will activate virtual environment.

2. install required dependencies by running 'pip install -r requirements.txt' in terminal.

3. Run Python3 data_extraction.py:
This script extracts data from the input.xlsx file and saves the processed information in the extracted_articles file.

4. Run Python3 text_analysis.py:
This script reads data from extracted_articles, performs calculations, and saves the results to the output_data_structure.xlsx file.
